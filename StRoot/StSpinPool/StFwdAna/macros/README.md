# StFwdAna Analysis Tools

This is a set of classes and files used for doing analysis with the STAR forward upgrade

- *StFwdData/* contains the basic data objects
  - *StFwdDataEvent* contains event level information
  - *StFwdDataFcs* contains FCS related data
	- *StFcsPhotonCandidate* represents FCS clusters/points
	- *StFcsPairCandidate* represent FCS cluster/point pairs

- *StFwdAna/* contains the analysis classes
  - *StFwdAnaVirtual* is the virtual base class from where all FwdAna analysis classes should inherit
  - *StFwdAnaData* contains the data in *StFwdData*
  - *StFwdAnaDataMaker* is the bridge between StMaker and the StFwdAnaVirtual analysis makers

- *runMuDst.C* ROOT macro to run the StFwdAna analysis classes on a STAR MuDst file

- *Merge.pl* can be used to merge ROOT files from job outputs

- *MakeRunList.pl* is a script to generate a run list for Run 22 production of the st_fwd stream data. Can be changed to find other productions as well

- *FilterPol.py* is a script used to filter the polarization data from the RHIC spin website
	- Generated by script above *Run22PolForJobs.txt* is a file that contains the polarization information for Run 22 and filtered for reading by the mini-makers

- *GetTrigList.csh* can be used to generate the FCS trigger list (requires DB access)
	- *FilterTriggers.py* is a script to filter the output from script above
		- Output of script above *FcsSortedTrig.txt* is a file that contains the trigger information for the FCS triggers in Run 22 and sorted and filtered for reading by the mini makers
